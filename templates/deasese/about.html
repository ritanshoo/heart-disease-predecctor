How can you use HDPS?





Anyone with a device having browser and internet can easily use Heart Disease Prediction System. If you want to predict heart disease using HDPS, all you need is to enter accurate value of 13 different attributes that system asks like age, sex, blood pressure and so on. HDPS will then return the chances of having heart disease in probability which resides from 0-1. In addition to that, if you want to save data for future purpose and want guidance on what to do next, you can simply sign up to the system which provides all those features.
